# 公文排版工具

> 将纯文本公文一键转换为符合《党政机关公文格式》国家标准（GB/T 9704-2012）的 Word 文档。

---

## 项目简介

公文排版工具是一个基于 PyQt5 的桌面 GUI 应用，专为党政机关、企事业单位公文写作场景设计。用户只需粘贴或导入纯文本公文，工具会自动解析文档结构（标题、各级标题、主送机关、正文、附件、署名、成文日期等），并按 GB/T 9704-2012 国标生成排版规范的 `.docx` 文件，省去手动调整字体、字号、行距、页码等繁琐步骤。

核心特色：

- **结构自动识别**：正则匹配一/二/三/四级标题、附件说明、主送机关、署名、成文日期
- **标点全角化**：自动将半角标点转换为全角，符合公文规范
- **标点字体分离**：所有标点符号（含各级标题内）统一使用仿宋_GB2312 字体
- **精确排版控制**：段前段后、首行缩进、行距均可配置，精确到 0.1 磅
- **国标默认参数**：开箱即用，无需手动调整即符合 GB/T 9704-2012
- **配置持久化**：设置自动保存，下次启动沿用
- **文本自动保存**：编辑内容自动留存，防止意外丢失
- **DOCX 导入**：支持导入已有 Word 文档继续编辑

## 默认参数（GB/T 9704-2012）

| 项目         | 默认值                                       |
| ------------ | -------------------------------------------- |
| 页边距       | 上 3.7 / 下 3.5 / 左 2.8 / 右 2.6 cm         |
| 页脚距       | 2.5 cm                                       |
| 公文标题     | 方正小标宋简体 二号 行距 38pt                |
| 正文         | 仿宋_GB2312 三号 行距 28.8pt 首行缩进 2 字符 |
| 一级标题     | 黑体 三号                                    |
| 二级标题     | 楷体_GB2312 三号                             |
| 三级标题     | 仿宋_GB2312 三号（加粗）                     |
| 段落缩进     | 文本之前/之后均为 0 字符                     |
| 段落间距     | 段前/段后均为 0 字符                         |
| 标点符号     | 全部使用仿宋_GB2312（含各级标题内）          |
| 标题与正文   | 之间空一行                                   |
| 页码字体     | Times New Roman 四号                         |
| 页码格式     | — 1 — 长横线分隔，奇右偶左                   |

## 使用方法

### 方式一：直接运行 EXE（推荐普通用户）

下载 `dist/公文排版工具.exe`，双击运行即可。无需安装 Python 环境。

### 方式二：从源码运行（推荐开发者）

```bash
# 环境要求：Python 3.10+
pip install PyQt5 python-docx mammoth

# 运行
python main.py
```

### 操作流程

1. 在编辑器中粘贴或输入公文文本（首行将作为公文标题）
2. 点击「设置」按钮调整排版参数（默认已符合国标，可直接跳过）
3. 点击「导出」生成 `.docx` 文件

## 技术栈

- **Python 3.14**
- **PyQt5** - GUI 界面
- **python-docx** - Word 文档生成（通过 OxmlElement 操作 w:spacing、w:ind、w:rFonts 等底层元素实现精确控制）
- **mammoth** - DOCX 文件导入
- **PyInstaller** - 打包为单文件 EXE
